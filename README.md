# 🛒 Flask E-Commerce Web Application

A full-stack eCommerce web application built using Flask, SQLite, and Stripe Checkout.

This project demonstrates backend development, database integration, session handling, and payment gateway integration.

---

## 🚀 Features

- Product listing from SQLite database
- Add to cart functionality
- Update quantity (+ / -)
- Remove items from cart
- Cart session management
- Stripe Checkout integration (test mode)
- Order success page
- Clean Git history with environment variable security

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- HTML / CSS
- Stripe API
- Git & GitHub
